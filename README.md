# Task 4 - Make a Website Mobile-Friendly Using CSS Media Queries

## What I Did
- Built a desktop-style webpage with fixed-width navbar, hero, about, cards, portfolio, and contact sections.
- Added working navigation — links scroll smoothly to their sections.
- Used a media query (`max-width: 768px`) to make the layout mobile-friendly: navbar and nav links stack vertically, sections become full-width, cards stack in a column.
- Tested responsiveness using Chrome DevTools and on a real mobile browser.

## Tools Used
HTML, CSS, Chrome DevTools, GitHub Pages

## Outcome
Learned how media queries and flexible layouts convert a desktop-only page into a mobile-friendly one.
